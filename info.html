환경 구현 설명

  1. vscode 좌측 extension 클릭
  2. ssh 검색
  3. Remote - SSH 설치
  4. 검색창 열기 => 윈도우 : f1 혹은 ctrl + shift + p, 맥북 : fn + f1
  5. 나오는 검색창에 'Remote-SSH: Open SSH Configuration File...' 클릭
  6. '.ssh/config'로 끝나는거 클릭
  7. 아래 파란글씨 복붙 후 저장
    8.         Host zeropointer
            HostName server.zeropointer.co.kr 
            User {{id}}
            Port 2022
  8. 4번 실행 후 'Remote-SSH: connect to Host' 클릭
  9. zeropointer 클릭
  10. 기다리기
  11. 비밀번호 입력
  12. 좌측 첫번째 아이콘(파일 모양) 클릭
  13. 좌측 Open folder
  14. 그냥 Enter
  15. 비밀번호 입력
  16. 'Yes'로 시작하는거 클릭

다운받아야할 프로그램(Extension)

  • C/C++
  • C/C++ Extension Pack
  • C/C++ Runner

비밀번호를 입력하고 싶지 않다면?

  1. 아래 파란색 모두 복사(좀 김) 
    2.             {{Pk}}
  2. 원하는 폴더에 파일이름을 'smart.pem'으로 저장
  3. 'smart.pem'의 경로 복사 => 윈도우 : 파일 우클릭 > 속성 > 위치 드래그 해서 복사 , 맥북 : 파일 우클릭 > 정보 가져오기 > 위치 드래그 해서 복사
  4. '.ssh/config'로 접속
  5. 위에 7번 과정을 아래 껄로 교체(경로는 3번 과정으로 얻은 경로) 
    6.             Host zeropointer
                HostName server.zeropointer.co.kr 
                User {{id}}
                Port 2022
                IdentityFile 경로/smart.pem
  6. 8번 과정과 똑같이 접속
